| [`kv-cache`](https://github.com/ajitashwath/kv-cache) | transformer key-value cache implementation from scratch |

| [`search-ranking`](https://github.com/ajitashwath/search-ranking) | pairwise learning-to-rank search engine served over http in go |

| [`neural-ode`](https://github.com/ajitashwath/neural-ode) | neural ordinary differential equations implemented in jax |
| [`pacman-ragent`](https://github.com/ajitashwath/pacman-rl/) | deep q-network (dqn) agent trained on atari pac-man using experience replay and target networks |
| [`smart-notes`](https://github.com/ajitashwath/smart-notes/) | local nlp pipeline generating summaries, key concepts, and exam questions from documents |
| [`web-crawler`](https://github.com/ajitashwath/multithreaded-web-crawler) | concurrent rust web crawler with robots.txt support, rate limiting, and async workers |
| [`scale-churn`](https://github.com/ajitashwath/scale-churn) | distributed churn prediction pipeline on hadoop |
| [`researcher`](https://github.com/ajitashwath/researcher) | multi-agent research system with analyst, writer, and reviewer agents generating complete reports |
