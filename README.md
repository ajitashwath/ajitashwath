# i build.
mostly backend systems and applied ai & ml infrastructure. the kind of stuff that runs in the background and either works or takes everything down with it.

| project | what it does |
|---|---|
| [`callmind`](https://github.com/ajitashwath/callmind) | voice agent platform that handles phone calls end-to-end, routes across llms, transcribes in real time |
| [`autonomous-ml`](https://github.com/ajitashwath/autonomous-ml) | mlops system that watches your model in prod, catches drift, and retrains itself without going down |
| [`prompt-compression`](https://github.com/ajitashwath/prompt-compression) | shrinks llm prompts by up to 70% using textrank + bart, tracks token delta per model |
| [`speculative-decoding`](https://github.com/ajitashwath/speculative-decoding) | speculative decoding with gpt-2 and distilgpt-2 as draft model |
| [`model-distillation`](https://github.com/ajitashwath/model-distillation) | resnet18 teacher → 43x smaller student, 4.3x faster inference |
| [`kv-cache`](https://github.com/ajitashwath/kv-cache) | key-value cache from scratch |
| [`kernel-suite`](https://github.com/ajitashwath/kernel-suite) | custom cuda kernels |
| [`search-ranking`](https://github.com/ajitashwath/search-ranking) | learning-to-rank search engine with a pairwise ranknet model, served over http in Go |
| [`streaming-token`](https://github.com/ajitashwath/streaming-token) | streaming token generation over websockets |
| [`neural-ode`](https://github.com/ajitashwath/neural-ode) | neural ordinary differential equationss in jax for modeling continuous-time dynamics |
| [`scale-churn`](https://github.com/ajitashwath/scale-churn) | churn prediction pipeline on hadoop |
| [`researcher`](https://github.com/ajitashwath/researcher) | multi-agent system with analyst, writer, reviewer agents generating full reports |
